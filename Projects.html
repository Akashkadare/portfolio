Tic Tac Toe Game

Tic Tac Toe is a classic two-player game played on a 3x3 grid. The players take turns marking a square with their chosen symbol: "X" or "O". The objective is to be the first to get three of your symbols in a row, either horizontally, vertically, or diagonally.

Weather Api

A weather API (Application Programming Interface) allows developers to access and use weather data from various sources. These APIs provide real-time and historical weather information, forecasts, and other meteorological data, enabling developers to integrate weather functionalities into their applications, websites, or services.
